# ClassicAssert.AreSame

**ClassicAssert.AreSame** tests that the two arguments reference the same object.

```csharp
ClassicAssert.AreSame(object expected, object actual);
ClassicAssert.AreSame(object expected, object actual,
               string message, params object[] params);
```
