# ClassicAssert.Contains

**ClassicAssert.Contains** is used to test whether an object is contained in a
collection.

```csharp
ClassicAssert.Contains(object anObject, ICollection collection);
ClassicAssert.Contains(object anObject, ICollection collection,
                string message, params object[] params);
```
