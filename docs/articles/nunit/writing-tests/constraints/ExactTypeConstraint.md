# ExactType Constraint

`ExactTypeConstraint` tests that an object is an exact Type.

## Constructor

```csharp
ExactTypeConstraint(Type)
```

## Syntax

```csharp
Is.TypeOf(Type)
Is.TypeOf<T>()
```
