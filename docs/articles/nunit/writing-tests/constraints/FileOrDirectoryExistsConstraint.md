# FileOrDirectoryExists Constraint

`FileOrDirectoryExistsConstraint` tests that a File or Directory exists.

## Constructor

```csharp
FileOrDirectoryExistsConstraint()
```

## Syntax

```csharp
Does.Exist
Does.Not.Exist
```

## Modifiers

```csharp
IgnoreDirectories
IgnoreFiles
```
