# InstanceOfType Constraint

`InstanceOfTypeConstraint` tests that an object is of the type supplied or a derived type.

## Constructor

```csharp
InstanceOfTypeConstraint(Type)
```

## Syntax

```csharp
Is.InstanceOf(Type)
Is.InstanceOf<T>()
```
