# ClassicAssert.AreNotSame

**ClassicAssert.AreNotSame** tests that the two arguments do not reference the same object.

```csharp
ClassicAssert.AreNotSame(object expected, object actual);
ClassicAssert.AreNotSame(object expected, object actual,
                  string message, params object[] params);
```
